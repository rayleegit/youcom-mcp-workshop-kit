# Lab 2: Inspect Source Pages

## Goal

Move from broad search to reading the source that actually matters.

## Prompt

```text
Use You.com search to find the most relevant official docs page for MCP web search.
Then inspect the page contents and summarize the setup steps in five bullets.
```

## Facilitator Notes

The point of this lab is to show that retrieval is a workflow, not a single call. Search gets you candidates. Page contents lets the assistant slow down and read the one that matters.
